Filtering Tasks:

Introduction:

Filters can significantly enhance productivity by allowing you to order/list tasks based on certain parameters.

If a tab includes a icon you can safely assume you are able to filter tasks within that tab.

You can click the icon to show the available filters and click the icon to hide the available filters.

Task Ticker offers four different filters: "Show Completed", "Filter by Priority", "Filter by Earliest", "Filter by Latest".

Note that not all tabs will include all four filters, some may include only three, others may include none.

This section will clarify the logic behind filters.

How Filters Works:

By applying "Show Completed" in eligible tabs you can view tasks that you already marked as completed; this filter works in conjunction with other filters.

By applying "Filter by Priority" in eligible tabs you can order the tasks in the tab by their priority; this filter can only work with the "Show Completed" filter.

By applying "Filter by Earliest" in eligible tabs you can order the tasks in the tab by their date, tasks with an earlier date will be shown higher up; this filter can only work with the "Show Completed" filter.

By applying "Filter by Latest" in eligible tabs you can order the tasks in the tab by their date, tasks with a later date will be shown higher up; this filter can only work with the "Show Completed" filter.

It should be noted that an "All-day" task (i.e., a task with a set date but without a specific time of the day) is considered as later than non "All-day" task. Moreover, tasks without dates will always be listed at the bottom when filtering by either "Earliest" or "Latest".

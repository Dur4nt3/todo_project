About Groups:

Introduction:

Groups allows users to organize related tasks into cohesive units, making it easier to track and manage them.

Whether its work-related tasks, event planning or personal productivity, groups allow you to improve your task organization.

To create a new group, simply assign the desired group name to a task with group capabilities, either by creating a new task with group capabilities, or by editing an existing task with group capabilities.

Listing Positions:

Listing positions refer to whether the group is displayed under the "My Groups" sub-section in the sidebar, and their order.

Depending on how many groups you have, you might find that between 0 and up to 5 groups are displayed in the sidebar.

Listing positions will always adjust according to the number of groups you have, that means, if you have 5 or more groups, you will always have 5 groups under the "My Groups" sub-section.

You may use the "Edit Groups" utility to change the listing order of groups to ensure that the desired groups are displayed on the sidebar.

To reiterate, you can NOT, display less groups than you have (up to a limit of 5) on the sidebar.

For the rest of the groups, you can find and access them via the "More Groups" utility.

Technical Details:

  • Group names can be up to 30 characters
  • Groups with no tasks are automatically deleted
  • Groups may be listed on the sidebar or in the "More Groups" interface
  • Group names may include letters, numbers, spaces or dashes
  • Group names may not include any other special characters
  • Groups may not share the same name
  • Groups may not use the name: "__unlisted__"
  • Groups may not have a listing position higher than 5

Editing Groups:

Hovering on the "My Groups" sub-section header on the sidebar will reveal a utility that allows you to edit groups (i.e., the aforementioned "Edit Groups" utility).

Clicking on said utility will reveal an interface that will allow you to customize groups.

The interface allows you to change a group's color label (i.e., the color of the "~" marker), change the group's name, and its listing order.

It should be noted that to change a group's name to an already existing name you will need to: find the group with the existing name and change its name to a unique one, submit the changes, and then, and only then, change the selected name to the initially desired name.

Unlisted Tasks:

You might encounter a scenario where you no longer want a task or a collection of tasks to be a part of any group.

In those cases, you can opt-out of the grouping utility and "un-list" tasks. You can achieve that by editing the task and selecting the "un-list" option.

Unlisted tasks will not be found under the "My Groups" sub-section, but can be found via the search utility. Specifically, click the "Search" tab and select the advanced search utility (by clicking the icon on the right side of the search bar), then search for tasks with the group name "__unlisted__".

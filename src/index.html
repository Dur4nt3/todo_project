Welcome to Task Ticker:

This help section includes a variety of resources created to assist you various questions or issues you might encounter.

Task Ticker offers the ability to create and monitor tasks.

Task Ticker has various types of tasks designed to fit your exact requirements.

Task Ticker allows you to create the following task types:

  • Basic
  • Grouped
  • Dated
  • Dated & Grouped
  • Repetitive
  • Repetitive & Grouped

Here's a quick run-down of the task types:

Basic tasks contain only the ability to set a task's title, description, and priority.

Grouped tasks contain all the functionality of a basic task, and the ability to set a task's group.

Dated tasks contain all the functionality of a basic task, and the ability to set a task's deadline.

Dated & Grouped tasks contain all the functionality of both a dated and grouped tasks.

Repetitive tasks contain all the functionality of a dated task, and the ability to make the task reoccur based on a set pattern.

Repetitive & Grouped tasks contain all the functionality of both a repetitive and grouped tasks.
